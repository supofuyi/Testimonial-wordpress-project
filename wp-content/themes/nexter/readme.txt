== Description ==

Bare WordPress Theme made for Block builder and Popular Page Builders.

This is completely blank theme with core WordPress structure. Everything can be build from Page Builders only. 


== Installation ==

1. In your site's admin panel, go to Appearance > Themes and click `Add New`.
2. Type “Nexter WordPress Theme in the search field.
3. Click `Install` and then `Activate` to start using the theme.
4. Navigate to Appearance > Customize in your admin panel and customize to your needs.

== Frequently Asked Questions ==

**Why this theme is blank? How to use this?**

We have made it blank intentionally to get maximum performance and compatibility with plugins related to block editor and other page builders. You can use block editor or any page builder such as Elementor Page builder and with that You may use The Plus Addons series of plugins to build every part of your WordPress website.
